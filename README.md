## PLH CodeData
Foi criado um repositório para organizar todos os trabalhos da Squad em um único local, utilizando a técnica de subtree do Git para integrar todos os projetos. Este é um projeto relacionado ao TCC do curso técnico de Desenvolvimento de Sistemas.
